# 서비스 개요: BakeryCrawler 인스타그램 수집기

## 1. 서비스 목표 (Service Objective)
제공된 디자인(`dashboard.html`, `add.html`)을 기반으로, 사용자가 타겟 베이커리 인스타그램 계정을 관리하고 최근 이슈(행사, 신메뉴 등)를 빠르고 깔끔하게 요약받을 수 있는 AI 기반 크롤링 대시보드 웹 어플리케이션입니다.

## 2. 디자인 및 테마 (Design System)
- **컨셉**: 따뜻하고 고급스러운 베이커리 테마 (Warm, premium bakery aesthetics)
- **주요 컬러**:
  - `primary`: `#8D6E63` / `#8B5E3C` (따뜻한 브라운)
  - `background-light`: `#FDFBF7` / `#F9F7F2` (부드러운 베이지/크림색 기준)
  - Dark mode 지원: 다크 모드 특화 서페이스 컬러와 텍스트 반전 처리
- **타이포그래피**:
  - `display`: Playfair Display, Work Sans (우아하고 현대적인 제목용)
  - `sans`: Noto Sans KR, DM Sans (가독성 높은 본문용)
- **UI 라이브러리**: Tailwind CSS, Material Symbols Outlined (아이콘)

## 3. 핵심 페이지 구성 (Core Pages)

### 3.1. 대시보드 메인 (`dashboard.html`)
- **초안/기능 요약**: 
  - **헤더/GNB**: 서비스 로고, API 사용량(예: 14/20), 최종 업데이트 시간, 다크모드/라이트모드 토글 액션 제공.
  - **사이드바 (관리 대상 계정 리스트)**: 
    - 활성 상태: 현재 수집 중인 베이커리 목록 (상태 뱃지: '활성', '동기화 중', '일시 중지됨' 등).
    - 수정/삭제 퀵 액션 라인버튼 존재.
    - 모니터링 통계 섹션: 전체 수집 게시물 수 및 요약 완료 건수 등 요약.
  - **메인 컨텐츠 (최신 AI 요약 정보)**:
    - 필터 및 새로고침 버튼 제공.
    - 단위 카드 형태: 원본 계정 프로필 이미지, 계정 이름, 최근 업데이트 시간 제공.
    - **카드본문 모듈**: AI가 분류한 핵심 정보가 모듈화되어 제공됨 (예: 신메뉴 소식, 예정된 이벤트, 일반 업데이트, 휴무 안내 위젯 등).
    - 원문 발췌 텍스트와 함께 '인스타그램 방문' 외부 링크 제공.

### 3.2. 새 타겟 등록 팝업/페이지 (`add.html`)
- **초안/기능 요약**: 
  - 신규 수집대상 인스타그램 계정을 시스템(`.tmp/target_accounts.json`로 관리 예정)에 추가.
  - **입력 폼**:
    1. 베이커리 이름 (`bakery-name`)
    2. 인스타그램 아이디/URL (`instagram-id`)
    3. 베이커리 카테고리 (`category` - 옵션: 아티장 브레드, 페이스트리/크루아상, 비건, 케이크/디저트, 베이글)
  - 등록 시 내부적으로 AI 에이전트가 이를 추적 대상으로 삼아 20개 API 제한 내에서 롤링 스크래핑을 시작.

## 4. 데이터 연동(Backend Context)
- **데이터 소스 제공**: Python Scripts (`scrape_instagram.py`, `summarize_posts.py`)가 백그라운드나 스케줄러(cron 등)를 통해 RapidAPI 크롤링 및 Gemini API 요약 결과를 JSON 구조체(`.tmp/summary.json`)로 떨궈줍니다.
- **프론트 연동 포인트**: Next.js 또는 정적 렌더링 프레임워크가 위 JSON을 받아들여, 위 모듈화된 카드와 리스트에 Hydration하여 사용자에게 시각적으로 보여줍니다.

## 5. 핵심 가치 (Core Value proposition)
- 사용자는 인스타그램의 불필요한 일상, 감성에 치우친 긴 원문을 일일이 파악할 필요 없이, 가장 중요한 '무엇을, 언제 파는지'라는 핵심 비즈니스 정보만 추출하여 볼 수 있습니다. 
- API의 타이트한 예산 한계를 시각적으로 투명하게 노출하여 효율적인 크롤링 타겟팅을 강제합니다.
